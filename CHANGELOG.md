### Added

- Experimental, security-focused MCP server over stdio and stateless Streamable HTTP, with mocked API integration coverage.
- Sandbox-first configuration, strict permission profiles, and pre-registration read-only filtering.
- Documented Sezzle v2 authentication, session, order, capture, refund, release, reauthorization, settlement, report, interest, and webhook adapters.
- Two-step mutation previews with explicit confirmation, state/request binding, idempotency, and audit evidence.
- Deterministic integer-money reconciliation, webhook verification/correlation, Integration Doctor, and secure support policy tools.
- Memory and SQLite storage, Docker/Compose, CI, Dependabot, resources, prompts, and security documentation.
